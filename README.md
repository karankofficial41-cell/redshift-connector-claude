# redshift-connector-claude

This redshift-claude-connector can be installed directly in Claude Desktop via **Install Extension**.

1. Open **Settings** → **Extensions**.
2. Click **Advanced settings** and find the **Extension Developer** section.
3. Click **Install Extension…** and select the `.mcpb` file.

Alternatives:
- Double-click the `.mcpb` file.
- Drag the `.mcpb` file onto the Claude Desktop window (not into a drop zone inside Settings).
